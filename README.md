
# Yamaha Dealership Billing Software

Desktop billing and inventory management software built for a two-wheeler dealership. Handles service billing, accessories billing, stock management, and PDF invoice generation.

## Features
- Service billing module
- Accessories billing module
- Stock/inventory management with data persistence
- PDF invoice generation
- Reports module
- Clean, easy-to-use desktop interface (Tkinter)
- Packaged as a standalone Windows executable

## Tech Stack
- **Language:** Python
- **GUI:** Tkinter
- **Database:** SQLite
- **Packaging:** PyInstaller (.exe)

## About This Project
This software was custom-built and deployed for a real two-wheeler dealership client, currently in trial use. It streamlines daily billing operations, tracks inventory, and generates printable invoices.
